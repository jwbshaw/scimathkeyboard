v0.452 current

  • iOS 26 gesture regression workaround forward-ported from v0.400 fork.
  • ImmediatePanGestureRecognizer fully integrated into main branch.
  • Key flash confirmed working post-fork merge.

v0.434

  • Slide-to-type removed. Gesture architecture conflicted irreconcilably with balloon gesture recognition.
  • Website updated to remove slide-to-type references.

v0.400 iOS 26 gesture fork

  • Apple regression (FB19394663 / FB19405284): UIPanGestureRecognizer no longer fires .began on stationary contact in keyboard extensions on iOS 26.x.
  • Workaround: ImmediatePanGestureRecognizer subclass overrides touchesBegan to force state = .began immediately.
  • Fork created to isolate and test workaround before forward-porting.
